#### mac终端git命令中英文切换

- 普通的命令行切换英文

  ```txt
  echo "alias git='LANG=en_GB git'" >> ~/.bashrc
  ```

  

+ 普通的命令行切换中文

  ```txt
  echo "alias git='LANG=zh_CN git'" >> ~/.bashrc
  ```

  

+ zsh的命令行切换英文

  ```txt
  echo "alias git='LANG=en_GB git'" >> ~/.zshrc
  ```



+ zsh的命令行切换中文

  ```txt
  echo "alias git='LANG=zh_CN git'" >> ~/.zshrc
  ```
